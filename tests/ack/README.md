# The Amsterdam Compiler Kit — Ack

Его github: https://github.com/davidgiven/ack
Его сайт: https://tack.sourceforge.net/

Запуск полученных файлов `test-c.com` и `test-pas.com` выполняется без
каких-либо проблем как в DOSBox, так и на железе с MS-DOS, включая вызов
процедуры на ассемблере. С запуском программы на Бейсике немного сложнее -
встроенному интерпретатору Бейсика требуется файл `tbas.bas.d`,
автоматически формируемый при компиляции, но в MS-DOS такое имя файла
некорректно. Поэтому нужно вручную переименовать файл, например, в
`tbas0bas.d` и заменить его же имя в COM-файле `test-bas.com`.
После этого запуск идет нормально, включая вызов процедуры.
